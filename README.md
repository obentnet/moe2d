## FAQ ##

### 账号密码问题 ###

    问：后台的登录密码以及账号忘记肿么办啊啊啊啊啊啊啊？

>   答：请前往程序根目录下的`confirm.ini.php`里查看账号密码设置。

### 如何重置系统 ###

    问：MD系统不想要啦！~想要重置！！！

>   答：请将`confirm.ini.php`里的内容清空即可。
